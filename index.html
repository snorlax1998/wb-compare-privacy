WB Compare+ — Политика конфиденциальности

Дата вступления в силу: 28 апреля 2026 г.

1. Назначение продукта

WB Compare+ — это расширение для браузера, которое помогает пользователям работать с карточками товаров Wildberries, отчётами, отзывами, удержаниями, полками рекомендаций и связанными аналитическими функциями.

2. Какие данные может обрабатывать продукт

  • технический идентификатор пользователя WB Compare+;
  • Telegram chat ID и username, если пользователь привязал Telegram-бота;
  • WB API-ключ, если пользователь добровольно добавил его для уведомлений и проверки удержаний;
  • данные, полученные из интерфейса Wildberries и открытых запросов сайта Wildberries, если это требуется для пользовательской функции расширения;
  • данные, полученные по запросам к внешним сервисам аналитики, если пользователь использует соответствующую функцию расширения;
  • служебные журналы, настройки уведомлений и технические статусы, необходимые для работы продукта.

3. Как используются данные

  • для отображения аналитики, сравнений, рекомендаций и пользовательских отчётов;
  • для проверки удержаний, штрафов и складских замеров по WB API;
  • для отправки уведомлений пользователю через Telegram;
  • для диагностики ошибок и поддержки пользователей;
  • для хранения пользовательских настроек и связки аккаунта.

4. Передача данных третьим лицам

WB Compare+ не продаёт пользовательские данные и не передаёт WB API-ключи третьим лицам для маркетинговых или рекламных целей.

Однако расширение и связанные серверные компоненты могут обращаться к внешним сервисам, необходимым для выполнения пользовательских функций, включая:

  • официальные API Wildberries;
  • Telegram Bot API для доставки уведомлений;
  • внешние аналитические сервисы, если пользователь использует функции, которые явно на них опираются.

5. Хранение и защита WB API-ключа

Если пользователь добавляет WB API-ключ, он хранится в зашифрованном виде. Ключ шифрования хранится отдельно от репозитория исходного кода. В интерфейсах администрирования и в логах открытое значение API-ключа не отображается.

6. Локальная обработка в браузере

Часть функций WB Compare+ работает локально в браузере пользователя. Однако некоторые функции требуют сетевых обращений к серверной части WB Compare+, к API Wildberries, к Telegram и к внешним аналитическим сервисам. Поэтому продукт не является полностью офлайн-локальным.

7. Удаление данных

Пользователь может запросить удаление связанных данных через поддержку. После удаления удаляются связанные настройки, Telegram-привязка и сохранённые ключи, если они присутствуют.

8. Какие данные не используются

WB Compare+ не использует пользовательские данные для продажи, рекламного таргетинга, брокерских целей или передачи неуполномоченным третьим лицам.

9. Контакты

По вопросам конфиденциальности и удаления данных используйте поддержку продукта или контактный адрес издателя, указанный в Chrome Web Store.
